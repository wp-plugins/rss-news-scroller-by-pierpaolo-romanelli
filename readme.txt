== Description ==
RSS News Scroller is a simple news scroller which you can use by just activating it. Classic jQuery scrolling effect attract user to read news from Feed.

* News Scroller with RSS
* Fade Effect
* Scrolling Effects settings
* Widget

== Installation ==
1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure the plugin by going to the RSS News menu that appears in your admin menu
